# 🎯 Objetivo Geral

O objetivo desta etapa é compreender como **sistemas lineares e invariantes no tempo (LTI)** processam sinais discretos, utilizando a **convolução** como ferramenta fundamental.

A atividade integra:

* fundamentos matemáticos
* análise de sistemas discretos
* implementação computacional
* interpretação prática de sinais reais

---

# 🧠 Abordagem do Projeto

O trabalho foi organizado em atividades progressivas:

* 📚 **Conceitos** → fundamentos teóricos de sistemas LTI  
* ✍️ **Cálculos manuais** → entendimento da convolução  
* 💻 **Simulações** → implementação no MATLAB/Octave  
* 📊 **Análises** → interpretação dos resultados  

Essa abordagem permite consolidar o aprendizado de forma teórica e prática.

# 🧩 Problema Norteador

> Como determinar a saída de um sistema discreto e garantir que ele seja estável, causal e adequado para aplicações reais?

### ✔️ Resposta obtida:

* a saída é obtida pela convolução entre entrada e resposta ao impulso  
* a análise de estabilidade e causalidade é essencial  
* filtros podem ser usados para melhorar sinais reais  

---

# ✅ Conclusão

Este estudo dirigido demonstrou, de forma integrada, como:

* a convolução permite prever o comportamento de sistemas LTI  
* equações de diferenças modelam sistemas discretos reais  
* a análise de estabilidade e causalidade garante funcionamento adequado  
* filtros digitais são ferramentas fundamentais na engenharia  

Assim, o estudante desenvolve uma compreensão sólida e aplicada do processamento digital de sinais.
