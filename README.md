# Processamento Digital de Sinais

Este repositório contém a implementação completa das atividades propostas na disciplina de **Processamento Digital de Sinais**, abordando desde os fundamentos teóricos até simulações computacionais e análise de resultados.

Todos os códigos foram desenvolvidos com foco didático, como um estudo dirigido da disciplina de Processamento Digital de Sinais, evitando dependências complexas e priorizando clareza na implementação dos conceitos.
